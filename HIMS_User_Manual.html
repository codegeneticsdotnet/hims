Welcome to HIMS User Manual

This interactive manual provides step-by-step instructions for the Hospital Information Management System (HIMS). Use the sidebar to navigate through different modules and workflows.

Quick Tip: Look for the colored badges to identify which user role is responsible for each action.
Secretary / Info Doctor Laboratory Billing Pharmacist

1. User Login

Accessing the system securely.

Open System ➜ Select Branch ➜ Enter Credentials

Steps:

  • Select your designated Branch from the dropdown menu.
  • Enter your registered Username and Password.
  • Click Login to access your dashboard.

2. Patient Entry Secretary

Creating a new patient record in the master database.

Dashboard ➜ Patient Management ➜ Add New Patient

Workflow:

  1. Log in as Secretary/Information user.
  2. Navigate to Dashboard > Patient Management.
  3. Click Add New Patient.
  4. Fill in required demographics (Name, Age, Contact, Address).
  5. Save to generate a unique Patient ID.

3. OPD Registration Secretary

Registering an existing patient for an Out-Patient Department visit.

Patient Entry ➜ OPD Module ➜ Add OPD Patient

Steps:

  • Ensure the patient is registered in the system (see Patient Entry).
  • Go to the OPD module.
  • Click Add OPD Patient.
  • Search for the patient by name or ID.
  • Encode the Attending Doctor and the patient's Medical Concern (Chief Complaint).
  • Save the registration.

4. OPD Patient Consultation

The clinical workflow involving Clinic Staff, Doctors, and Billing.

A. Clinic Assessment Clinic Staff

Path: OPD Registration > Clinic Dashboard
1. Select the patient from the list.
2. Encode Vital Signs (BP, Temp, Weight, etc.).
3. Upload any previous Image Records if available.
4. Proceed to Billing or Doctor.

B. Doctor Consultation Doctor

Path: OPD Registration > Doctor Dashboard
1. Select patient from queue.
2. Review history and vitals.
3. Click Diagnostics to request labs (if needed).
4. Click Add Diagnosis to record findings.
5. Mark as Discharge / Bill to finish consultation.

C. Billing Billing / Pharmacist

Path: Discharge / Bill > Billing > POS
1. Navigate to Billing module.
2. Select OPD to Bill list.
3. Open POS (Point of Sale) for the patient.
4. Process payment for Consultation Fee and other charges.

5. OPD Laboratory Services

Handling lab requests, payments, and result processing.

A. Request Creation Laboratory

Path: OPD Registration > Laboratory Dashboard
1. Select patient.
2. Click Create Request.
3. Select tests to be performed.
4. Advise patient to proceed to Lab Payment.

B. Payment Billing

Path: Billing > POS
1. Cashier opens patient's bill.
2. Collects payment for Lab Request.
3. System updates status to "Paid" / "Process Request".

C. Process Request Laboratory

Path: Laboratory Dashboard > Search Patient
1. Verify payment status.
2. Perform tests.
3. Update Status to Active (Processing) and then Done (Results Ready).

6. OPD Pharmacy Sales / Walk-In Pharmacist

Dispensing medication for walk-in patients or OPD prescriptions.

POS ➜ Search Items ➜ Payment

Steps:

  • Open the POS module.
  • Search Items: Type drug name or generic name.
  • Add items to cart (quantity and dosage).
  • Process Payment: Accept cash/card.
  • Print Receipt and Dispense items.
